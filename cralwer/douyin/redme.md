抖音爬虫工具
项目简介
这是一个基于Python的抖音数据爬虫工具，能够自动化获取抖音视频、作者信息及评论数据。该工具采用模块化设计，包含链接获取和内容爬取两大部分，支持无头模式运行，适合数据采集和分析需求。
安装要求
依赖库
bash
pip install DrissionPage
系统要求
Python 3.7+
支持Windows、Linux和macOS系统
需要安装Chrome浏览器

使用方法

1. 克隆或下载本项目
2. 安装依赖库：pip install DrissionPage
3. 运行主程序：python Douyincrawler.py
4. 按提示选择搜索模式（关键词搜索或目标用户主页）
5. 输入关键词/用户名
6. 程序将自动获取链接并爬取数据，结果保存在output目录中
文件结构

├── douyinurlget.py # 视频链接获取模块
├── douyincrawler.py # 数据爬取模块
├── Douyincrawler.py # 主程序
└── output/ # 输出文件目录
数据输出
爬取的数据将按类型保存在output目录中：
视频内容：content_{视频ID}_{日期}.csv
作者信息：author_{视频ID}_{日期}.csv
评论数据：comment_{视频ID}_{日期}.csv
二级评论：subcomment_{视频ID}_{日期}.csv
搜索结果：{关键词}_search_results.csv 或 {关键词}_user_videos.csv
注意事项
1. 请遵守抖音平台的使用条款，合理控制爬取频率
2. 该工具仅用于学习和研究目的，请勿用于商业用途
3. 滚动次数(scroll_times)参数控制获取视频数量，次数越多获取的视频越多，但耗时也越长
